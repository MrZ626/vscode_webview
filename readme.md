# VS Code Webview

> **声明：该项目是我（MrZ_26）的第一个实验性 Vibe Coding 项目，所有技术细节几乎均由Claude Opus 4.6完成**  

## 项目动机

我想在安卓设备上使用 VS Code 作为文本（代码）编辑器，外出可以只带键盘不用带笔记本电脑

用浏览器打开的话可能会有快捷键冲突，那我想单独打包成一个app就能避免

## 项目功能

webview打开vscode.dev链接，没了

然后由于不是浏览器环境，光一个webview并无本地文件读写功能，那这菜过头了肯定不行，于是让AI把打开文件（夹）请求做了桥接，实现了基本的文件读写功能

注：文件和文件夹的增删改还是有点工作量的，AI也犯错修改了好几轮
